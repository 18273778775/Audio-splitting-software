音频分割工具 v3.1 - 安装使用指南
=====================================

🎉 欢迎使用音频分割工具 v3.1 高精度版！

## 📦 快速安装

### 1. 下载文件
- 文件名：音频分割工具.exe
- 文件大小：约192MB
- 系统要求：Windows 10/11 (64位)

### 2. 安装步骤
1. 将 "音频分割工具.exe" 下载到您的电脑
2. 建议创建专门的文件夹存放（如：C:\音频工具\）
3. 双击 "音频分割工具.exe" 启动程序

### 3. 首次运行
- 首次启动可能需要10-15秒加载时间
- 如果系统询问防火墙权限，请选择"允许"
- 部分杀毒软件可能误报，请添加到信任列表

## 🚀 快速上手

### 基本操作（3步完成）
1. 📁 点击"浏览"按钮选择音频文件（支持MP3、WAV）
2. ⚙️ 选择分割模式并设置参数
3. ▶️ 点击"开始分割"按钮

### 分割模式选择

#### 🔸 固定时长分割
- 适用场景：均匀分割音频
- 设置方法：输入时长（如：60秒）
- 示例：将1小时音频分割成60个1分钟片段

#### 🔸 自定义长度分割
- 适用场景：按特定需求分割
- 设置方法：输入长度序列（如：3,5,10）
- 示例：第1段3秒，第2段5秒，第3段10秒，第4段剩余

#### 🔸 智能分割（可选）
- 功能：在静音处寻找最佳分割点
- 设置：勾选"启用智能分割"
- 搜索范围：建议1-2秒

## 📋 使用示例

### 示例1：播客分割
```
文件：podcast.mp3 (30分钟)
模式：固定时长
设置：5分钟
结果：生成6个5分钟片段
```

### 示例2：音乐分段
```
文件：song.mp3 (4分钟)
模式：自定义长度
设置：30,60,90
智能分割：启用
结果：前奏30秒，主歌60秒，副歌90秒，尾奏40秒
```

### 示例3：语音分析
```
文件：speech.wav (10分钟)
模式：固定时长
设置：30秒
智能分割：启用（2秒搜索范围）
结果：在句子间隙处分割，每段约30秒
```

## ⚠️ 注意事项

### 使用前准备
✓ 确保有足够的磁盘空间（至少是原文件的2倍）
✓ 备份重要的原始音频文件
✓ 关闭其他占用内存的程序

### 常见问题解决
❓ 程序启动慢？
   → 正常现象，首次加载需要时间

❓ 内存占用高？
   → 根据音频文件大小自动调整，属正常现象

❓ 分割失败？
   → 检查文件路径是否包含特殊字符
   → 确保有足够的磁盘空间
   → 尝试使用较小的文件测试

❓ 输出文件找不到？
   → 检查原文件目录下的"output"文件夹
